Grid Layout

Boot strap comes loaded with a well built grid system to allow me to create content that looks great on any device. The grid layout creates these boxes of content that rearrange themselves when on a larger or smaller device.

Responsive

Responsive is a key word in web development. It means that the website responds to its environment so content is always easy to access and looks great. Bootstrap is built off the idea of being responsive.

Try it out!
